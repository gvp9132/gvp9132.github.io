变量
变量就是存储数据(值)的地方，变量名就是存储数据的容器,变量可以在不知道具体值得时候提前使用
  • 注意 :   变量的声明要放在所有执行代码的最前面
  • 变量类型 :   String(字符类型) , Integer(数字类型)
  • 变量值为String时需要使用""包裹, 只有声明为Integer的值才能进行数学运算
  • 变量声明 :   Dim 变量名 As 变量类型 
                范例: 
                        Sub test()
                            Dim name As String ' 定义(声明)变量name ,类型是字符
                            Dim age As Integer ' 定义(声明)变量age ,类型是数字
                        End Sub
  • 变量赋值 :   变量名 = 需要设置的变量值
                  name = "张三" '设置变量name的值是 张三
  • 变量值修改 :   变量名 = 需要修改的值
                  name = "李四" '修改变量name的值为 李四
  • 变量的赋值和修改形式一致, 第一次设置变量值叫赋值,之后再设置变量值叫修改 , 设置变量值得时候一定是变量名在左变量值在右
  • 完整例子: 
                                {
                                Sub test()
                                Dim name As String ' 定义变量name
                                Dim age As Integer ' 定义变量age
                            
                                name = "张三" ' 为变量设置值
                                age = 18 '为变量设置值
                            
                                [A1] = name  ' 使用变量name的值,修改A1单元格的值为变量name的值
                                [A2] = age ' 使用变量的值,修改A2丹阳的值为变量age的值
                            
                                name = "李四" '修改变量的值
                                age = 20 '修改变量的值
                            
                                [B1] = name '使用变量name修改之后的值,设置到B1单元格
                                [B2] = age ' 使用变量age修改之后的值,色值到B2单元格
                                End Sub
                            }
    以上代码执行完毕后excel表格如下,代码中的(单引号)'表示注释,vba程序会忽略'后面的内容:
    # A B
    1 张三 李四
    2 18 20
    1
